# הכנה למבחן בכימיה אורגנית 🧪

אתר לימוד בעברית (RTL) לקראת מבחן בכימיה אורגנית, בנוי ב-Next.js. כולל 12 נושאי לימוד מלאים, כל אחד עם:

- סיכום עם דגש על ההבדלים בין מנגנונים/חומרים (תיבות "הבדלים מרכזיים")
- טבלאות השוואה
- תרשימי זרימה/מנגנון (mermaid)
- מודלים תלת-ממדיים אינטראקטיביים של מולקולות מרכזיות (שליפה חיה מ-PubChem + 3Dmol.js)
- שאלון מעורב (רב-ברירה, נכון/לא נכון, השלמה, סידור שלבים)
- משחק התאמה (חיבור קווים בין מבנה/מושג לשם/הגדרה) בחלק מהנושאים

בנוסף יש נושא נומנקלטורה נפרד (עם משחק התאמה נוסחה↔שם) וטאב "הפסקת התרעננות" עם תמונות כלבים/חתולים אקראיות שמתעדכנות, תצוגה מוגדלת (lightbox) והורדה.

האתר רספונסיבי למחשב, טאבלט (פורטרט/לנדסקייפ) ונייד (פורטרט/לנדסקייפ).

## הרצה מקומית

```bash
npm install
npm run dev
```

פתחו [http://localhost:3000](http://localhost:3000).

## מבנה תוכן

- `src/content/types.ts` - טיפוס `Topic` המשותף לכל נושא.
- `src/content/topics/*.ts` - קובץ תוכן אחד לכל נושא (סיכום, טבלאות, תרשימים, מולקולות, שאלון, משחק התאמה).
- `src/content/moleculeCids.ts` - מיפוי שמות מולקולות ל-CID מאומת מול PubChem, לצורך המודלים התלת-ממדיים.
- `src/components/` - קומפוננטות משותפות (Quiz, MatchingGame, Molecule3DViewer, MermaidDiagram, RefreshBreak וכו').
- `src/app/api/pets/route.ts` - proxy צד-שרת לתמונות כלבים/חתולים (dog.ceo / thecatapi), כדי לעקוף מגבלות CORS בדפדפן.

## פריסה ל-Vercel

1. אתחלו מאגר Git (כבר בוצע, ראו `git log`) ודחפו ל-GitHub:
   ```bash
   git remote add origin <כתובת המאגר שלכם ב-GitHub>
   git push -u origin main
   ```
2. היכנסו ל-[vercel.com/new](https://vercel.com/new), חברו את חשבון ה-GitHub, ובחרו את המאגר.
3. Vercel יזהה אוטומטית שמדובר בפרויקט Next.js - אין צורך בהגדרות נוספות (`npm run build` הוא פקודת ה-build המובנית).
4. לחיצה על Deploy - האתר יעלה תוך דקה-שתיים.

לחלופין, לפריסה ישירה מהמחשב בלי GitHub, ניתן להשתמש ב-Vercel CLI:
```bash
npx vercel
```
